# SmartSite API - Gestione Documentale Ingegneristica

Backend per un sistema gestionale documentale (DMS) orientato all'ingegneria, sviluppato in Laravel.

## 🚀 Funzionalità Completate
- [x] Architettura Multi-Tenant sicura (Isolamento dati aziendali).
- [x] Database con UUID per massima sicurezza.
- [x] Autenticazione API tramite Token (Laravel Sanctum).
- [x] Gestione anagrafica documenti.

## 🛠️ Stack Tecnologico
- PHP 8.x / Laravel 11
- MySQL
- Laravel Sail (Docker)
- Laravel Sanctum (Autenticazione)
